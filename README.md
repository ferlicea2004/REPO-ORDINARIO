# Ordinario

Este repositorio contiene los proyectos educativos/ejemplo en las carpetas `Android/` y `React/`.

Instrucciones rápidas:

- Inicializar localmente (ya hecho):
  - `git init`
  - `git add .`
  - `git commit -m "Initial import of Ordinario (Android + React)"`
- Crear remote en GitHub y subir (opciones):
  - Con GitHub CLI (si estás autenticado):
    - `gh repo create <usuario>/<repo-name> --public --source=. --remote=origin --push`
  - Manualmente en GitHub: crea el repo y luego:
    - `git remote add origin https://github.com/<usuario>/<repo>.git`
    - `git branch -M main`
    - `git push -u origin main`

Si quieres, puedo crear el remote por ti (necesito autorización/token) o puedo ejecutar los comandos de push si me das la URL del repo remoto.
